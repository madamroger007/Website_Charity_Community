## :toolbox: Getting Started

### :bangbang: Prerequisites

- Install Python<a href="https://www.python.org/"> Here</a>
- Read documentation flask<a href="https://flask.palletsprojects.com"> Here</a>


### :gear: Installation

This project uses venv python as package manager
```bash
python3 -m venv venv
```
Next you must activate your venv example in OS Linux :
```bash
. ./venv/bin/activate
```
Next you must activate your venv example in OS Window:
```bash
\venv\Scripts\activate
```


### :triangular_flag_on_post: Deployment
